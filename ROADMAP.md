# Roadmap — Koras.Results

The full product roadmap with feature specifications lives in [docs/product/product-roadmap.md](docs/product/product-roadmap.md) and [docs/features/future-roadmap.md](docs/features/future-roadmap.md).

## Summary

| Version | Theme | Highlights |
|---|---|---|
| 0.1.0-preview.1 | Core preview | Core + AspNetCore packages, CI, docs skeleton |
| 0.1.0 | Full preview | All five packages, full test suite, samples, docs |
| 0.5.0 | API freeze candidate | Feedback-driven refinements, benchmark tuning, migration guides |
| 1.0.0 | Stable | Frozen public API, compatibility policy in force |
| 1.1.0 | Ecosystem | EF Core exception mapping, localization providers, LINQ query syntax |
| 1.2.0 | Protocol edges | gRPC status conversion, GraphQL error conversion |
| 2.0.0 | Source generation | Error catalogs, AOT-certified mappers |

## Out of scope (permanently)

Option/Maybe/Either types · validation rule engines · mediator implementations · HTTP client concerns · retry policy engines.

Feature requests: please open a GitHub issue. Roadmap items are commitments of intent, not dates.
